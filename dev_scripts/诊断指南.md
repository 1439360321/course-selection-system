# 选课系统诊断指南

> 最后更新: 2026-06-05 | 系统已完成全部功能测试

## 常见问题速查

| 现象 | 根因 | 解决 |
|------|------|------|
| 页面表格完全空白 | table id ≠ `bootstrap-table` | RuoYi `$.table.init()` 默认 ID 为 `bootstrap-table`，自定义 ID 需覆盖 `id` 参数 |
| 安全审计登录异常显示 `--` | `$.get`(GET) 调用了 `@PostMapping` | 统一为 `@GetMapping("/stats")` |
| 学生评教无法打分 | 列表页缺"操作"列，无法跳转评分页 | 添加"去评教"按钮 + 控制器返回已评/未评混合列表 |
| 教师提交开课申请失败 | FK 约束 `fk_course_admin` 拒绝教师号 + insertCourse 缺 status/maxStudents | 删 FK + 补字段 |
| 管理员看不到待审核课程 | CourseMapper.xml 未 SELECT status 列 + WHERE 无 status 过滤 | 补全 selectCourseVo、resultMap 和 WHERE 子句 |
| 成绩申诉"同意/驳回"点击无效 | `$.modal.prompt` 在 RuoYi 中不存在 | 改为 `layer.prompt({title,formType:2}, callback)` |
| 课程评价 tno 为 NULL 插入失败 | 课程无授课教师记录 | 兜底用 course.adminId + 返回错误提示 |
| 教师/学生登录后看不到侧边栏 | SysLoginController 重定向到独立页面 | 统一返回 `homeUrl: "index"` |
| URL 参数获取失败（sno/tno） | 模板中 JS 读 URL query string 但侧边栏 tab 模式下无参数 | 改为 Thymeleaf `[[${sno}]]` / `[[${tno}]]` 注入 |
| `$.modal.prompt` is not a function | RuoYi 未实现此方法 | 统一用 `layer.prompt()` |

- ✅ 公共: 消息中心 + 个人设置 + BCrypt密码
