# 选课系统使用手册

## 一、系统概述

基于 **若依 RuoYi v4.8.3** 开发，Spring Boot 4.0.3 + MyBatis + Apache Shiro + Thymeleaf + MySQL 8.x。
支持三种角色：**管理员**、**教师**、**学生**，RBAC 权限隔离。

## 四、数据库设计（12 张业务表）

| 表 | 说明 | 关键字段 |
|------|------|------|
| `department` | 院系 | dept_id PK, dept_name, address |
| `admin` | 管理员 | admin_id PK, password_hash (BCrypt) |
| `student` | 学生 | sno PK, sname, dept_id FK, password_hash (BCrypt) |
| `teacher` | 教师 | tno PK, tname, rank, dept_id FK, password_hash (BCrypt) |
| `course` | 课程 | cno PK, cname, credit, course_type, max_students, status (审核状态) |
| `class` | 授课关联 | (tno,cno) PK, semester, class_time, exam_time |
| `courseselection` | 选课记录 | (sno,cno) PK, semester, normal_score, test_score |
| `audit_log` | 审计日志 | log_id PK, action, detail, ip_addr |
| `course_notice` | 课程公告 | notice_id PK, cno FK, tno FK, title, content |
| `course_evaluation` | 学生评教 | eval_id PK, sno FK, cno FK, rating(1-5), comment |
| `grade_appeal` | 成绩申诉 | appeal_id PK, sno FK, cno FK, reason, status, reply |
| `notification` | 消息通知 | notif_id PK, recipient_type, recipient_id, title, is_read |

**高级特性**: 3 触发器（级联删除）+ 4 视图 + 1 存储过程 (sp_student_gpa)

## 六、安全体系

| 类别 | 措施 | 实现 |
|------|------|------|
| 密码 | BCrypt 加密 | 替代 MD5，`$2a$` 格式，旧密码登录时自动迁移 |
| 防暴破 | 登录锁定 | 5 次失败锁 10 分钟 (Ehcache TTL=600s) |
| 验证码 | 数学计算 | `CaptchaValidateFilter`，登录页显示算式 |
| XSS | 过滤器 | `XssFilter` 覆盖全部业务路径 |
| 审计 | AOP 切面 | `@SecurityLog` 注解自动记录 IP/操作/目标 |
| 监控 | 安全仪表盘 | 登录失败数 + 实时事件列表 |
| 日志 | 登录 + 操作 | `sys_logininfor` + `sys_oper_log` |
| CSRF | 框架支持 | Shiro `CsrfValidateFilter`（当前演示关闭） |

## 七、课堂演示流程（15 分钟）

**1. 数据库展示（2min）** — Navicat 展示 12 张表 + 触发器 + 视图 + 存储过程
> 亮点：总评=平时×40%+考试×60%，实时计算不存储（3NF）；BCrypt 加密

**2. 管理员（4min）** — 院系管理 → 课程审核(通过/驳回) → 统计分析(图表+安全概览) → 选课设置

**3. 教师（4min）** — 开课申请 → 发公告 → 录入成绩 → 导出Excel → 查看评教 → 处理申诉

**4. 学生（3min）** — 选课(容量校验) → 看公告 → 评教 → 申诉 → 学分统计 → 下载PDF成绩单

**5. 安全亮点（2min）** — BCrypt密码(`$2a$`) + 登录锁定演示 + 验证码 + 审计日志 + XSS防护

## 八、部署

```bash
mysql -u root -p course_selection < deploy_package/sql/course_selection.sql
for f in RuoYi/sql/{bcrypt,course_audit,course_notice,course_evaluation,grade_appeal,notification,selection_window,test_data_enhancement}_migration.sql; do
  mysql -u root -p course_selection < "$f"
done
cd RuoYi && mvn clean package -Dmaven.test.skip=true
java -jar ruoyi-admin/target/ruoyi-admin.jar
# http://localhost:8080
```
