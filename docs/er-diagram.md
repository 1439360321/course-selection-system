# ER 图（第二版）

## 实体（5 个）

| 实体 | 属性 |
|---|---|
| **学生** Student | 学号、姓名、性别、出生日期、籍贯、手机号 |
| **教师** Teacher | 工号、姓名、性别、出生日期、职称、薪资 |
| **课程** Course | 课程号、课程名、学分、学时 |
| **学院** Department | 院系编号、院系名称、地址、电话 |
| **管理员** Admin | 工号、姓名 |

## 联系（8 个）

```
            管理员
             │
             ├── 管理学生 (1:N) ─→ 学生
             ├── 管理教师 (1:N) ─→ 教师
             └── 审核开课 (1:N) ─→ 课程

    学院
     │ 1
     │ 归属
     │ N
  ┌──┴──────┐
 学生       教师
  │N         │N
  │          │
  │  选课    │  授课
  │  M:N     │  M:N
  │          │
  └────┬─────┘
       课程
```

| # | 联系 | 基数 | 说明 |
|---|---|---|---|
| 1 | 学院 **归属** 学生 | 1:N | |
| 2 | 学院 **归属** 教师 | 1:N | |
| 3 | 学院 **开设** 课程 | 1:N | |
| 4 | 学生 **选课** 课程 | M:N | 自身属性：**学期**、平时成绩、考试成绩、总评成绩 |
| 5 | 教师 **授课** 课程 | M:N | 自身属性：**学期**、上课时间 |
| 6 | 管理员 **管理** 学生 | 1:N | 管理员管理学生账号 |
| 7 | 管理员 **管理** 教师 | 1:N | 管理员管理教师账号 |
| 8 | 管理员 **审核** 开课 | 1:N | 管理员审核教师提交的开课申请 |

> 选课联系包含属性：学期、平时成绩、考试成绩、总评成绩
> 授课联系包含属性：学期、上课时间
