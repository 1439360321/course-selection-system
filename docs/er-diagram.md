# ER 图

## 实体（5 个）

| 实体 | 属性 |
|---|---|
| **学生** Student | 学号, 姓名, 性别, 出生日期, 籍贯, 手机号, 密码(Bcrypt) |
| **教师** Teacher | 工号, 姓名, 性别, 出生日期, 职称, 薪资, 密码(Bcrypt) |
| **课程** Course | 课程号, 课程名, 学分, 学时, 类型(必修/选修), 最大容量, 审核状态 |
| **学院** Department | 院系编号, 院系名称, 地址, 电话 |
| **管理员** Admin | 工号, 姓名, 密码 |
