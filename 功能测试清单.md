## 🧪 测试用例

### 测试1：课程管理功能
```sql
-- 查询课程列表
SELECT * FROM course LIMIT 5;

-- 检查max_students字段是否存在
SELECT cno, cname, max_students FROM course LIMIT 3;
```

### 测试2：选课功能
```sql
-- 查询选课记录
SELECT * FROM courseselection LIMIT 5;

-- 统计选课人数
SELECT cno, COUNT(*) as enrolled FROM courseselection GROUP BY cno;
```

### 测试3：用户登录验证

-- 检查管理员账号

## ✅ 已修复的问题

| 问题 | 原因 | 修复方法 |
|------|------|----------|
| 中文乱码 | SQL文件GBK编码导入到UTF-8数据库 | 使用Python脚本正确转换编码 |
| 缺少max_students字段 | 数据库表结构与代码不一致 | ALTER TABLE添加字段 |
| 检查约束错误 | GBK编码的中文字符无法识别 | 修复检查约束为UTF-8字符 |
