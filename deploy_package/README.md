# 选课系统 - 部署说明

## 📦 部署包内容

```
deploy_package/
├── bin/
```

## 🛠️ 环境要求

### 必需软件
1. **JDK 17 或更高版本**
   - 下载地址: https://www.oracle.com/java/technologies/downloads/
   - 或使用 OpenJDK: https://adoptium.net/

2. **MySQL 8.0 或更高版本**
   - 下载地址: https://dev.mysql.com/downloads/mysql/

### 推荐配置
- Windows 10/11
- 内存: 4GB 以上
- 硬盘: 500MB 可用空间

---

## 📝 部署步骤

### 第一步：安装JDK
1. 下载并安装 JDK 17
2. 配置环境变量（可选，也可以直接修改启动脚本）
   - `JAVA_HOME` = JDK安装路径（如: `C:\Program Files\Java\jdk-17`）
   - 在 `Path` 中添加: `%JAVA_HOME%\bin`

### 第二步：安装并配置MySQL
1. 下载并安装 MySQL 8.0
2. 设置 root 用户密码（推荐设为: `123456`）
3. 记住安装路径，确保 MySQL 服务正在运行

### 第三步：初始化数据库
1. 双击运行 `初始化数据库.bat`
2. 如果 MySQL 配置不同，请编辑脚本中的变量：
   ```batch
   set MYSQL_HOST=localhost
   set MYSQL_PORT=3306
   set MYSQL_USER=root
   set MYSQL_PASSWORD=123456
   set DATABASE_NAME=course_selection
   ```
3. 等待脚本执行完成，看到"数据库初始化完成！"提示

### 第四步：启动系统
1. 如果 JDK 没有配置环境变量，请编辑 `启动系统.bat`
2. 修改 `JAVA_HOME` 为实际的 JDK 路径
3. 双击运行 `启动系统.bat`
4. 等待启动，看到"若依启动成功"提示

### 第五步：访问系统
- 打开浏览器，访问: **http://localhost:8080**
- 默认端口: 808080

## ⚙️ 配置说明

### 修改数据库连接
如果需要修改数据库连接配置，请编辑 jar 包中的 `application-druid.yml`：

1. 使用压缩软件打开 `bin/ruoyi-admin.jar`
2. 找到 `BOOT-INF/classes/application-druid.yml`
3. 修改数据库配置
4. 保存并放回 jar 包

### 修改端口
默认端口是 80，如果被占用可以修改：

1. 打开 jar 包
2. 找到 `BOOT-INF/classes/application.yml`
3. 修改 `server.port`
4. 保存

## 🚩 常见问题

### Q: 启动时提示"未找到Java环境"
A: 请检查 JDK 是否安装，并修改 `启动系统.bat` 中的 `JAVA_HOME` 路径

### Q: 数据库连接失败
A: 
1. 确认 MySQL 服务正在运行
2. 检查数据库用户名和密码是否正确
3. 确认数据库名称是 `course_selection`

### Q: 访问 http://localhost:8080 无法打开
A:
1. 确认系统已成功启动（看到"若依启动成功"）
2. 检查 8080 端口是否被占用
3. 尝试修改端口号（见上方"修改端口"）

### Q: 如何停止系统？
A: 双击运行 `停止系统.bat`，或在启动窗口按 `Ctrl+C`
